# next-forge-paddle

**Production-grade Turborepo template for Next.js apps.**

[next-forge-paddle](https://github.com/danbillson/next-forge-paddle) is a [Next.js](https://nextjs.org/) project boilerplate for modern web application forked from [next-forge](https://github.com/haydenbleasel/next-forge). It is designed to be a comprehensive starting point for new apps, providing a solid, opinionated foundation with a minimal amount of configuration using Paddle for payments and billing.

Clone the repo using:

```sh
npx next-forge-paddle@latest init
```

Then read the [docs](https://docs.next-forge.com) for more information.
